指针类型

类型名简介
Box指向类型T的，具有所有权指针，有权释放内存
&T指向类型T的借用指针，也称引用，无权释放内存，无权写入数据
&mut T指向类型T的mut形借用指针，无权释放内存，有权写入数据
*const T指向类型T的只读裸指针，没有生命周期信息，无权写入数据
*mut T指向类型T的可读写裸指针，没有生命周期信息，有权写入数据
在标准库还有一种封装起来可以当智能指针使用的类型，叫”智能指针“）
类型名简介
Rc指向类型T的引用计数指针，共享所有权，线程不安全
Arc只想类型T的原子型引用计数指针，共享所有权，线程安全
Cow<’,T>Copy-on-writer，写时复制指针。可能是借用指针，也可能是具有所有权的指针
