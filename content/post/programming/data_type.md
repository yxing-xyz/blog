## C

```
                        整形: short int long

                数值类型:
                        浮点类型: float double

基本数据类型:

                字符类型: char
空类型(void)

指针类型

构造数据类型: 数组、结构体(struct)、共同体(union)、枚举(enum)
```

## GO

```
数值类型:        uintptr (u)int (u)int8 (u)int16 (u)int32 (u)int64
                float32 float64 complex64 complex128
                byte 别名 uint8
                rune 别名 int32

字符串:          string

布尔类型：        bool

派生类型:
  初始值不为nil:  array、struct
  初始值nil:      pointer、function、interface、slice、map、channel
```

## JavaScript

```
Number、String、Boolean、Null、Undefined、Symbol 引用类型(Object、Array、function)
```

## Python

```
Number(数字): int、float、bool、complex(复数)
String(字符串)
List(列表)
Tuple(元组)
Set(集合)
Dictionary(字典)
```

## Java
```
基本数据类型：byte、short、int、long、char、double、boolean
引用类型：类类型、接口类型、数组、null
```

## PHP
```
标量：整数、浮点数、布尔、字符串
复合：数组、对象
特殊：资源、null
```
